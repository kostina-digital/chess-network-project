# Features

## Authentication

- User registration
- Login and logout
- Password reset request and reset flow
- Cookie-based authenticated session handling

## Profiles

- Public user profile pages
- Editable profile fields:
  full name, username, bio, rating, favorite opening
- Avatar upload with validation
- Default generated avatars for new users

## Social Graph

- Follow and unfollow users
- Followers and following counters
- Modal lists for followers and following

## Posts

- Create posts with title and content
- Attach up to three images per post
- Edit existing posts, including image management
- Feed and profile post views
- Missing post images fall back to a bundled 404 illustration

## Engagement

- Like posts
- Add comments to posts
- Like or dislike comments
- Comments show author link and relative timestamp

## Search

- Search for users by username or full name
- Search for posts by title or content
- Combined search results page

## News

- Curated chess news listing
- Detail page per article
- Pagination

## UX and UI

- Consistent typography helpers
- Responsive layouts across primary pages
- Authenticated header search, centered; layout constrained to 1440px max width
- Reusable cards and modal patterns
- Unified red styling for all error messages
